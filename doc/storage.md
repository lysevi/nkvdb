* категории. могут добавляться в процессе инициализации хранилища
* кеш данных. текущие значения и небольшая история.
* поддержка записи в прошлое и будущее
* поиск по индексу
* поддержка запросов на поиск минимума/максимума, интерполяции, интеграла... любой обработки данных


Параметры:
* глубина хранения данных - как далеко в прошлое можно писать данные
* максимальный размер страницы
* размер кешированных данных в памяти.

Создание хранилища:
1. Создание папки. Если уже есть, то очистка и потом создание.
2. Создание первой страницы.

Открытие
1. Получение списка страниц.
2. Поиск той, что более старая.
3. Подгрузка самой старой.
